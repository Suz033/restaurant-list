# Restaurant List

一個在地美食網，可用名稱或類別搜尋想要的餐廳

點進卡片可看到餐廳詳細介紹，包含聯絡電話、評分等等，點擊地圖旁圖示還可快速打開 Google Map 查看位置

## Installing

1. 安裝套件

```
npm install
```

2. 安裝 Nodemon 至全域 (若有安裝過可省略)
```
npm install -g nodemon
```

3. 啟動！(`ctrl+c` 可停止)
```
npm run dev
```

4. 請參考 `.env.example` 格式添加 `.env` 檔案

5. 可添加種子資料 (`ctrl+c` 可停止)
```
npm run seed
```
